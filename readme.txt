Sistema de CATEGORIES - - - - POSTS - - - -TAGS

Obs.

Criar as tabelas com o SQL abaixo:
bookmarkers.sql
Alterar no banco a relação entre categorias e posts para CASCADE
Fazer um CAKE BAKE para todas as tabelas 
Fazer as alterações como no link abaixo:
http://book.cakephp.org/3.0/pt/quisckstart.html

ATENÇÃO:
Para chamar a função tagged usar a fórmula:
localhost-ou-seu-dominio/projeto/controler/tagged/nome-da-tag
